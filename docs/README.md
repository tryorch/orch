# Orch Docs

This is the public documentation site for Orch, built with Astro Starlight.

[![Built with Starlight](https://astro.badg.es/v2/built-with-starlight/small.svg)](https://starlight.astro.build)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The root `md/` directory contains working notes and design docs. Public docs should live here under `docs/src/content/docs`.
